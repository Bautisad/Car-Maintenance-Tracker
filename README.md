# Vehicle Maintenance Tracker

A full stack web app that tracks user's vehicle maintenance. Also recommends the required services based on mileage.

## Tech Stack Used

- Frontend: Next.js, React, TypeScript, Tailwind CSS
- Backend: Java Spring Boot
- Database: PostgreSQL
- Tools: Git, Github, Postman

## Planned Features

- Add and manage vehicles
- Log vehicle maintenance history
- Update vehicle current mileages
- Recommend services based on mileage intervals
- View overdue and upcoming services
